## Getting started
### Backend
Run a server.
```
cd backend
go run .
```
The mysql server is automatically connected to the lab's server and the api server is set up on the university's local network.

### Frontend
```
cd frontend
```
start app
```
npm run dev
```
mock api
```
npm run json server 
```
