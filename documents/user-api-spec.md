## Register user
Request:
* Endpoint: /api/register
* Method: POST
* Request Body:
  * username: username (string)
  * password: password (string)
Response:
* Success Response:
  * HTTP Status Code: 201 Created
  * Response Body:
    * message: wip (string)
* Error Responses:
  * HTTP Status Code: 400 Bad Request
  * Response Body:
    * error: wip (string)
